Open command line

* Install node.js locally on your machine (check on cmd by running "npm -v" and "node -v")
* Clone project locally by running "git clone https://gitlab.com/miloshevski/Internship-angular.git" on cmd
* Navigate on the project dir in order to execute following statements
* Run "npm install" in order to install locally all the node modules (in case if they are not installed)
* Run "npm start" in order to run application locally (if it doesn't start up go to the next step)
* You might neet to install local server as a npm library (Run "npm install http-server -g" on your command line)
* Navigate on http://localhost:8000 on your browser
